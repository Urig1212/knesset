# משחק הקואליציה

משחק הרכבת קואליציה בעברית, ממותג מקור ראשון, עם גרירה של מפלגות, חוקים הניתנים להפעלה, תמיכה מבחוץ והתאמה למובייל.

## פיתוח

```bash
npm install
npm run dev
```

## בנייה

```bash
npm run build
```
